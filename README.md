# node-dns-window
[![Build with Panto][build-image]][build-url] [![Build Status][travis-image]][travis-url]

DNS tool, built by electron.

![screenshot](http://ww4.sinaimg.cn/large/801b780ajw1f8oegzzb23j208c0got9a.jpg)

```sh
npm i panto-cli -g
npm i
npm run build
npm start
```
