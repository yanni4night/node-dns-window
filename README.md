# node-dns-window

DNS tool, built by electron.

![screenshot](http://ww3.sinaimg.cn/large/801b780ajw1f8o5b49nvqj20go0xcq49.jpg)

```sh
npm i panto-cli -g
npm i
npm run build
npm start
```
